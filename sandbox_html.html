Executive Summary

Introduction

XYZ Corp. is happy to present the following proposal to ABC Inc. The high-level services identified at this time are as follows:

  • Define a software specification for the ABCD Software
  • Implement the ABCD Software as per the specification
  • Perform QA, performance analysis and testing of the ABCD Software prior to delivery

The key features of the ABCD Software are:

  1. Support both current in-school PCs (with IE 8 and later browsers) and iPads.
  2. A custom Flash-based multi-media interactive web-application that will run on modern browsers and allow kindergarten students to practice and improve their reading skills. This will use custom graphic resources along with animation and audio to provide feedback.
  3. A HTML5 web-application that will provide the same functionality as the Flash-based web-application but will run on iPads.
  4. A student tracking and monitoring module that will keep track of each student's progress while using the software. A custom API will be developed so that the above two and also future applications will be able to provide statistics to the tracking and monitoring module.
  5. A classroom management application that will allow teachers and administrators to manage the students, classes and other features of the software.

ABCD Software

Initial Screens

These are the initial screens that will be commonly used to access the software:

  1. Student Login Screen: serves as a welcome screen in which the student selects their name from a list of students in the class.
  2. Select Letter Screen: contains a map or other custom interface to allow the student to select a letter to advance to the activities for that letter.
    • The current week's letter is highlighted.
  3. Letter Screen: shown after a letter is selected. The Letter Person appears and an associated song is played. The Letter Person greets the child and shares his or her special characteristic.
    • This screen also includes some spoken audio, such as instructions, feedback and sounds/words when letters/objects are clicked.
